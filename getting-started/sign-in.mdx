Use the Affretio authentication page to enter an existing workspace or create the
first Owner account for a new workspace.

![Affretio sign-in page with Log in and Create workspace tabs](/images/authentication/sign-in.jpg)

## Sign in

<Steps>
  <Step title="Open Affretio">
    Open the workspace URL supplied by Trucklab or your workspace Owner.
  </Step>
  <Step title="Select Log in">
    Keep **Log in** selected and enter the email address associated with your access.
  </Step>
  <Step title="Enter your password">
    Enter your current password. Use **Show password** only when nobody else can see your screen.
  </Step>
  <Step title="Open the workspace">
    Select **Log in**. Affretio opens the workspace associated with your session.
  </Step>
</Steps>

## Create the first workspace

Use this flow only when your organization does not already have an Affretio workspace.
Creating a second workspace does not add you to an existing one.

1. Select **Create workspace**.
2. Enter the workspace name, Owner email address, and a password of at least eight characters.
3. Add the Owner's first and last name when available.
4. Select **Create workspace**.
5. Complete the [initial setup checklist](/getting-started/initial-setup).

<Warning>
  If colleagues already use Affretio, ask an Owner to create your access from
  **Settings → Users**. Do not create another workspace to recover access.
</Warning>

## Reset a forgotten password

1. Select **Forgot password?** on the sign-in page.
2. Enter your Affretio email address.
3. Select **Send reset link**.
4. Open the reset link you receive.
5. Enter a new password of at least eight characters and select **Update password**.
6. Return to **Log in** and use the new password.

<Note>
  A reset page without a valid token cannot update a password. Start the reset
  request again if the link is missing, incomplete, or expired.
</Note>

## End a session

Select **Log out** at the bottom of the sidebar when you finish using a shared or
unattended computer. Closing a tab is not a substitute for signing out.
